# JU Heroes Training — Тренировки Героев

Веб-игра для семьи: квесты на физические упражнения, образовательные модули, система достижений и семейный рейтинг. Работает как PWA — можно установить на телефон.

## Компоненты

- `index.html` — основная страница с экранами: логин, дашборд, квесты, образование
- `app.js` — логика приложения: авторизация, квесты, XP/уровни, достижения, пуш-уведомления
- `education.js` — модуль образовательных квестов (AI-грамотность)
- `style.css` — стили: glassmorphism, анимации, адаптивная вёрстка
- `icons.js` — кастомные SVG-иконки для квестов
- `sw.js` — Service Worker для PWA и пуш-уведомлений
- `manifest.json` — PWA-манифест

- PWA (installable)
- GitHub Pages (хостинг)

## Запуск

Статический сайт — открой `index.html` или запусти локальный сервер:

```bash
npx serve .
```

Или просто перейди на: https://viacheslavzhuk-source.github.io/ju-heroes-training/

# Changelog

## [v1.2.0] - 2026-03-25

### Added
- ElevenLabs TTS озвучка — 9 MP3 аудиофайлов для образовательных квестов (замена браузерного speechSynthesis)
- Видео-квесты для 20 образовательных модулей (YouTube IFrame Player API)
- Прогресс-бар просмотра видео, разблокировка квеста после 80%
- Защита паролем — PIN-модалка при входе для каждого члена семьи
- Система автообновлений — network-first для кода, тост при новой версии
- PWA иконки 192x192 и 512x512, регистрация Service Worker при загрузке
- Скилл YouTube для загрузки видео через API
- Скилл Vimeo (документация API)

### Changed
- Service Worker: network-first стратегия для HTML/JS/CSS (всегда свежий код)
- Service Worker: cache-first для аудио/картинок (не меняются)
- Автопроверка обновлений каждые 30 минут
- manifest.json: относительные пути, корректная работа на GitHub Pages

### Fixed
- PWA не устанавливалась — отсутствовали иконки и SW не регистрировался при загрузке
- manifest.json start_url не работал на GitHub Pages (абсолютный путь)

## [v1.1.0] - 2026-03-24

### Added
- Авторизация по пользователям — 4 предустановленных профиля (Данил, Лев, Папа, Мама)
- Раздельный прогресс для каждого пользователя в localStorage
- Экран выбора пользователя с карточками и статистикой
- Семейный рейтинг — реальные данные всех участников
- Кнопка смены пользователя (иконка профиля в навбаре)
- Пуш-уведомления (Service Worker + Notification API)
- Напоминания о тренировке (если сегодня не тренировался)
- Уведомления о выполнении квеста, достижениях, повышении уровня, серии
- In-app баннер с напоминанием
- Панель настроек уведомлений (колокольчик в навбаре)
- PWA-манифест (можно установить на телефон)
- Service Worker с кэшированием файлов

### Changed
- Убрана форма ручной регистрации (имя/возраст/аватар)
- Рейтинг теперь показывает только реальных семейных участников

### Removed
- Фейковые лидеры в рейтинге (Артём, Маша, Дима, Лиза, Макс)

## [v1.0.0] - 2026-03-24

### Added
- Первый релиз
- Квесты: турник, отжимания, брусья, пресс

- Система XP и уровней
- Достижения
- Загрузка видео
- Glassmorphism UI с анимациями
